# STY003 — Four-space indentation

## What it checks
The file uses four spaces per indentation level (no tabs, no two-space
nesting).

## Why
Matches the SigmaHQ canonical style and avoids visual ambiguity around
nested selectors.

## Bad example
```yaml
detection:
  selection:           # two-space indent
    Image: x
  condition: selection
```

## Good example
```yaml
detection:
    selection:
        Image: x
    condition: selection
```

## How to fix
Reformat with a YAML-aware tool (`yamlfmt`, `prettier --parser yaml`).
