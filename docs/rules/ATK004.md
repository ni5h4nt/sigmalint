# ATK004 — Sub-technique specificity heuristic

## What it checks
If a parent technique is cited and a sub-technique exists that more
precisely describes the behavior, prefer the sub-technique.

## Why
Sub-technique tags drive accurate coverage reporting and dashboards.

## Bad example
```yaml
tags: [attack.t1059]      # Command-Line Interpreter (parent)
```

## Good example
```yaml
tags: [attack.t1059.001]  # PowerShell
```

## How to fix
Pick the most specific sub-technique that matches the detection.
